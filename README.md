# 极简选区扫码 (Minimalist Selection QR Scanner)

一款专为 Edge/Chrome 优化的浏览器扩展，通过“用户选区”精准识别网页上的二维码。

### ✨ 特色功能
- **选区识别**：支持手动划定区域，完美解决超长网页海报中二维码识别难的问题。
- **稳定性增强**：针对 Edge 垂直标签模式进行了专项优化，避开环境锁定报错。
- **自动化流**：解码后自动复制到剪贴板，并弹出可点击的通知直接跳转链接。
- **隐私保护**：全本地像素处理，不上传任何数据到服务器。

### 🛠️ 安装方法
1. 下载本项目并解压。
2. 打开 Edge 浏览器，进入 `edge://extensions`。
3. 开启左下角的“开发人员模式”。
4. 点击“加载解压缩的扩展”，选择本项目文件夹。

极简选区扫码也可以在edge商店add-ons里搜索到直接安装
